## 移植原因：

1. chrome插件需要翻墙（有门槛）;
2. 初版有很多不完善的功能;
3. 到目前为止原版一直没有更新;
本插件经过自己和同事几个版本的迭代用起来已经和实用；目的就是为了更好的更方便;
